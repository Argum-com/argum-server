# Argum Server

## Requirements

- [Rust](https://www.rust-lang.org/tools/install)

## Usage

```bash
cargo run
```

## Environment Variables

For a list of environment variables, see the [`.env.example`](.env.example) file.

| Variable | Description       | Default |
| -------- | ----------------- | ------- |
| `PORT`   | Port to listen on | `3000`  |
